# slot-machine-simple

C++で実装した、コンソール上で遊べるシンプルなスロットマシンです。

## 概要

所持金をクレジットに換金し、スロットマシンを回して所持金を増やすゲームです。

- 開始所持金: `1000`
- クリア条件: 所持金とスロット内クレジットの合計が `10000` 以上
- 1回のベット: `50`
- リール数: `3`
- 表示行数: `3`

メッセージは日本語で表示されます。状態メッセージはANSIカラー付きで、次の3種類に分類しています。

- `Info:` 通常の状態通知
- `Warning:` 入力ミスや所持金不足などの注意
- `Error:` 通常起きない内部状態エラー

絵柄名は `Cherry`, `Bell`, `Seven`, `Bar`, `Lemon` のように英語表記のままです。

## ビルド

`clang++` を使う場合:

```bash
clang++ -std=c++17 -Wall -Wextra -Wpedantic main.cpp -o slot-machine
```

`g++` を使う場合:

```bash
g++ -std=c++17 -Wall -Wextra -Wpedantic main.cpp -o slot-machine
```

## 実行

```bash
./slot-machine
```

## 遊び方

ロビーでは次のコマンドを使います。

```text
s: スロット開始
q: ゲーム終了
```

スロット中は次のコマンドを使います。

```text
s: スピン
e: スロット終了
q: ゲーム終了
```

スロット開始時に、所持金からクレジットへ換金する金額を入力します。クレジットがベット額より少なくなると、自動でスロットから退出します。
